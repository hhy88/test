## 1. 产品概述

MarkEdit 是一款本地轻量化 Markdown 编辑阅读一体化 Web 客户端，1:1 复刻 Typora 最新正式版全部原生能力。覆盖极简写作、学术公式排版、技术文档绘图、批量文件管理、全格式合规导出、自定义主题样式、全场景快捷键办公，全流程贴合原版操作习惯，老用户零学习成本无缝迁移。

- 核心目标：所见即所得实时编辑，取消传统分栏双面板，单页实时双向联动，输入语法即刻渲染成品样式
- 目标用户：Typora 现有用户、技术写作者、学术研究人员、文档工程师
- 产品价值：提供与 Typora 完全一致的编辑体验，同时基于 Web 技术实现跨平台统一体验

## 3. 核心流程

### 3.1 文档编辑主流程

用户打开应用 → 选择/新建MD文件 → 进入WYSIWYG编辑模式 → 输入Markdown语法实时渲染 → 切换源码模式查看/编辑原始语法 → 保存/导出文件

### 3.2 文件管理流程

用户打开文件夹 → 文件树递归展示 → 点击文件切换编辑 → 外部修改监听弹窗提示 → 拖拽移动/重命名/删除操作

### 3.3 导出流程

用户触发导出 → 选择格式 → 配置参数（页边距/主题/编码等） → 进度条展示 → 成功/失败提示

```mermaid

    "进入编辑器主界面" --> "WYSIWYG编辑模式"
    "进入编辑器主界面" --> "源码模式"
    "WYSIWYG编辑模式" --> "输入Markdown语法"
```

## 4. 用户界面设计

### 4.1 设计风格

- 主色调：浅色主题纯白底 #FFFFFF + 深色主题 #1e1e1e，强调色 #4285f4
- 按钮风格：扁平化圆角按钮，hover 微底色变化，点击轻压反馈
- 字体：正文 16px / 1.6 行高，标题逐级递减，代码块等宽字体
- 布局风格：经典桌面编辑器三栏布局，左侧可折叠面板 + 中间编辑画布 + 顶部菜单栏
- 图标风格：线性图标，16px 尺寸，与文字对齐

## 5. Markdown 语法支持清单

### 5.1 基础语法

| 语法类型 | 语法格式 | 渲染效果 |
|----------|----------|----------|
| 六级标题 | # ~ ###### | 字号逐级递减，加粗权重 |
| 加粗 | **text** | 粗体文字 |
| 斜体 | *text* | 斜体文字 |
| 删除线 | ~~text~~ | 删除线文字 |
| 行内高亮 | ==text== | 高亮底色文字 |
| 上标 | ^text^ | 上标文字 |
| 下标 | ~text~ | 下标文字 |
| 键盘按键 | ++key++ | 键盘按键样式 |
| 有序列表 | 1. item | 自动序号递增 |
| 无序列表 | - item | 圆点/方块/短横线三样式 |
| 任务列表 | - [ ] / - [x] | 复选框实时勾选 |
| 引用块 | > text | 左侧竖线嵌套引用 |
| Alert提示块 | > [!NOTE] | 四类风格提示块 |
| 分隔线 | *** / --- / ___ | 细灰实线 |
| 表格 | \| \| | 可视化编辑表格 |
| 行内代码 | `code` | 灰底等宽字体 |
| 代码块 | \`\`\`lang | 语法高亮代码块 |
| 行内公式 | $formula$ | MathJax实时渲染 |
| 块级公式 | $$formula$$ | 独立居中公式 |
| 链接 | [text](url) | 可点击链接 |
| 图片 | ![alt](url) | 自适应宽度图片 |
| 脚注 | [^1] | 悬浮预览脚注 |
| Mermaid图表 | \`\`\`mermaid | 流程图/时序图/甘特图等 |
